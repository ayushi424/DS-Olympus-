This folder consists of everything that is needed for instagram algorithm analysis.
It contains the required dataset as well as jupyter notebook containing analysis.

The analysis is regarding the reach of instagram posts using data of various fetaures such as likes, comments, followers, shares and hastags.
Visualisations have been provided with different features so as to see the behaviour.

It also has wordclouds of words used in post captions as well as hashtags to analyse the reach of the post accordingly.
